# K8s::Client

Ruby client library for the Kubernetes (1.10) API

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'k8s-client'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install k8s-client
